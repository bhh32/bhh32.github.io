Clean Your Room is a converted project that I started in Unreal Engine 4. I made the decision to convert it over because it seemed like more of a Unity project than it was suited for Unreal. It's target audiance is kids ages 3 - 11, but it's also fun to play for adults! I mean, who doesn't like a good explosion and gold stars?!? This is a pet project of mine because it's mainly being developed for my 6 (almost 7) year old daughter, and she's enjoyed testing for me its entire lifetime of development. It can be played as both a mouse/keyboard game or Xbox One controller game. I created a system that auto detects if you have an Xbox controller connected. If it detects one it defaults to the controller, whereas if it doesn't detect an Xbox controller its controls are mouse/keyboard.

I used Kenny Assets for the character and props in the room. Other interesting features this game has are as follows:
- Camera system that detects if the player is close enough to a wall the wall becomes transparent so the player can still be seen.
- Pickup inheritance: Both types of pickups come from a base pickup script that is then extended by any other pickup objects that have special features.
- Exploding Pickups: The exploding pickups spawn more base pickups that are then added into the achievement system as to how many pickups need to be collected before getting the next achievement.
- Achievement System: This is created to populate the star UI as the player collects more pickups, it's recalculated when the user hits an exploding pickup. However, if an achievement is earned it isn't taken away just because an exploding pickup was hit.
